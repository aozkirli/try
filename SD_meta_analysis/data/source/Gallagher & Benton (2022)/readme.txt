Data for the binary choice task are included in all_binary_choice_data.csv
This file includes participant number, stimulus orientation in degrees, participant response (1 = left, 0 = right), response time in seconds, noise level of stimulus.
Each row contains these data for a single trial.

Data for the adjustment task are included in all_adjustment_task_data.csv
This file includes participant number, stimulus orientation in degrees, participant response in degrees, noise level of stimulus.
Each row contains these data for a single trial.

noise levels are 0.5 or 4 and indicate the concentration parameter of a von Mises distribution (see Gallagher and Benton 2022 for details).
